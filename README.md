# ZARA Sales Analysis
## Project Overview
This project provides a comprehensive analysis of ZARA's sales data using Excel for data cleaning, PostgreSQL for advanced querying, and Power BI for dynamic visualizations. The dashboard helps uncover sales patterns, product performance, seasonal trends, and effective promotional strategies for improved decision-making.

## Tools Used
Excel: Data cleaning, preprocessing, and initial exploration

PostgreSQL: Data aggregation, advanced SQL queries, and ETL

Power BI: Data modeling, dynamic dashboards, and visualization

## Key Insights :
Total Sales Volume: 22 million units across all product categories

Top Product: Jackets, with over 12 million units sold

Promotional Impact: Promotions drive more than 53% of sales transactions

Seasonal Trends: Autumn yields peak sales, followed by steady declines through winter, spring, and summer

Sourcing Origins: Major sources include Bangladesh, Cambodia, and China for stock products

## Data Source
Real sales and inventory data from ZARA (mock or anonymized for demonstration).

## Dashboard Features
Sales by Product, Promotion and Season

Price Analysis by Category

Top Sales Origins Table

Actionable Recommendations for Marketing & Inventory

## How to Use
Clean and preprocess raw data in Excel.

Upload cleaned data into PostgreSQL for advanced querying.

Connect PostgreSQL to Power BI for visualization, applying DAX and data modeling as needed.

Explore interactive dashboard for strategic insights.

## Recommendations
Targeted promotions and seasonal marketing

Personalized CRM driven customer engagement

Enhanced stock management during high and low demand periods
